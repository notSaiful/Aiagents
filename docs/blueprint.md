# **App Name**: Gemini Notes

## Core Features:

- Note Input: Simple textarea input with a clear call to action to transform the note.
- AI Summary Generation: Use the Gemini API to transform pasted notes into an aesthetic summary with bullet points.
- Flashcard Generation: Leverage Gemini as a tool to generate flashcards (Q&A JSON) from the input text for review.
- Mind Map Creation: Use Gemini as a tool to create a mind map JSON structure from the pasted notes for enhanced visualization.
- Output Display: Display transformed outputs (summary, flashcards, mind map) within a tabbed interface for easy navigation.
- Share Functionality: Implement a share button to facilitate easy sharing of the generated content.
- User Authentication: Integrate Firebase Authentication for user management (Google + Email).

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calm and intellectual feel.
- Background color: Light beige (#F5F5DC) for a clean, pastel aesthetic.
- Accent color: Pale rose (#FFB6C1) to highlight interactive elements and create visual interest.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Minimalist layout focusing on mobile-first design with generous use of whitespace.
- Subtle transitions and animations to enhance user experience without overwhelming the interface.
- Use clean, line-based icons to maintain a clutter-free and modern aesthetic.
